# 25 + 5 Clock (Pomodoro Timer)

A modern Pomodoro Timer built with React for the FreeCodeCamp Front End Development Libraries Certification. This application helps users improve productivity by alternating between focused work sessions and short breaks.

https://github.com/solinda-cheab/fcc-25-5-clock

## Features

-  25-minute work session timer
-  5-minute break timer
-  Start / Pause timer
-  Reset timer to default values
-  Increase or decrease session length
-  Increase or decrease break length
-  Audio notification when a timer ends
-  Automatically switches between Session and Break
-  Responsive design
-  Passes all FreeCodeCamp user stories

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fcc-25-5-clock.git
```

Navigate to the project folder:

```bash
cd fcc-25-5-clock
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## FreeCodeCamp Requirements

This project fulfills all required FreeCodeCamp user stories, including:

- Adjustable session length (1–60 minutes)
- Adjustable break length (1–60 minutes)
- Start / Pause functionality
- Reset functionality
- Countdown displayed in `mm:ss` format
- Automatic transition between Session and Break
- Audio notification when time reaches `00:00`
- Required element IDs implemented
- Passes the FreeCodeCamp test suite
